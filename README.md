# Kuramoto synchronization model for Unity

## Installation
Released as a UPM package on OpenUPM.

- Add URL "https://package.openupm.com" in a Scoped Registry
- Add scope "jp.nobnak"
- Add package "Kuramoto model" in Package Manager.
